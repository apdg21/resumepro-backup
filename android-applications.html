Voco — Voice Type Smart

Floating microphone — speak, copy, paste anywhere

🚀 Finalizing for Production

Next-gen voice typing. Adolf built Voco as a floating mic overlay that follows the user across every app. A single tap starts listening — recognized speech is instantly copied to the clipboard, ready to paste anywhere. Clean, lightweight, and fully privacy-focused with no cloud dependency.

✨ Core Experience

🖱️ Floating Mic

Draggable overlay, works over any app

📋 Smart Paste

Auto-copies recognized text for instant pasting

🔊 Natural Language

Android speech recognizer with system locale

🧩 Lightweight

Minimal battery impact, single-toggle service

📱 What users can do

Take notes hands-free Dictate messages Quick voice search Fill forms eyes-free
Technology Stack
FloatingView Service WindowManager Overlay SpeechRecognizer Intent ClipboardManager Kotlin Coroutines Drag-to-move

Implementation: Core service built from FloatingVoiceService.kt, MainActivity.kt, SettingsActivity.kt, and SpeechRecognitionActivity.kt — fully functional floating mic with voice recognition.

📦 Preparing for Play Store Production release ~ Q2 2025
📝⏰

RemindMe — Notes & Reminder

Voice-first notes with smart reminders

🧪 Closed Testing (Play Console)

No typing required. RemindMe is Adolf's fully functional productivity app centered around voice-first note creation. Users speak their notes, set reminders, and let the app handle everything — backed by a Room database, swipe gestures, and reliable alarm-based notifications.

✅ Implemented Features

🎤 Voice-to-Text Notes

Create notes by speaking — no keyboard needed

⏰ Smart Reminders

AlarmManager + reliable notifications

🗂️ Local Database

Room with DAO & entity relations

↔️ Swipe to Delete

With undo support

🎨 Material Design

Dark/light theme support

🔍 Search & Filter

Quick note lookup

Technology Stack
Kotlin Room ViewModel/LiveData AlarmManager Notifications RecyclerView
