Voco - Voice Type Smart

Floating microphone – speak, copy, paste anywhere

🚀 Finalizing for Production

Next‑gen voice typing: A floating mic button that follows you across any app. Tap to speak, get instant text copied to clipboard and ready to paste. Built from the Kotlin sources you provided – clean, lightweight, and privacy‑focused.

✨ Core Experience

🖱️ Floating Mic

Drag anywhere, tap to start listening – works over any app

📋 Smart Paste

Recognized text is auto‑copied; one tap to paste where you need it

🔊 Natural Language

Uses Android speech recognizer with system locale

🧩 Lightweight Service

Minimal battery impact, starts/stops with a single toggle

FloatingView Service WindowManager Overlay SpeechRecognizer Intent ClipboardManager Kotlin Coroutines Permission handling Drag-to-move

Take notes hands‑free Dictate messages Quick search by voice Fill forms eyes‑free
