‚úèÔ∏è Editing Your Resume

How do I edit my resume content?

Use the included form.html file to easily update your resume information through a user-friendly interface. The form will generate a properly formatted data.json file for you.

Can I edit the resume directly in the JSON file?

Yes! Advanced users can edit the data.json file directly. Make sure to maintain the proper JSON structure and validate your changes using JSONLint.

How do I change the profile photo?

Replace the assets/profile.jpg file with your own photo (200x200 pixels recommended). Keep the filename exactly the same.

Can I customize the colors and fonts?

Absolutely! Edit the CSS variables in style.css to change the color scheme. Modify the font-family properties to use your preferred fonts.

üåê Hosting Guide

How do I host my resume on Netlify?
  1. Sign up for a free account at Netlify.com
  2. Drag and drop your entire resume folder into the Netlify Drop zone
  3. Your resume will be instantly available at a URL like: your-site.netlify.app
How do I use GitHub Pages?
  1. Create a new GitHub repository
  2. Upload all your resume files to the repository
  3. Go to Settings > Pages and enable GitHub Pages for the main branch
  4. Your resume will be available at: username.github.io/repository-name
Can I use my own domain?

Yes! Both Netlify and GitHub Pages allow you to connect a custom domain. You'll need to:

  1. Purchase a domain from a registrar (Namecheap, Google Domains, etc.)
  2. Follow the platform-specific instructions for DNS configuration
  3. Set up SSL for secure HTTPS connections

üí° Resume Tips

What makes a great digital resume?
  • Concise content - Keep descriptions brief but impactful
  • Clear hierarchy - Make important information stand out
  • Mobile-friendly - Test on different devices
  • Fast loading - Optimize images and assets
  • Professional appearance - Use consistent styling
How should I write my work experience?
  • Start with action verbs (Developed, Led, Implemented)
  • Focus on achievements rather than duties
  • Use numbers when possible ("Increased sales by 30%")
  • Keep each bullet point to 1-2 lines
  • Tailor to the job you're applying for
What skills should I include?
  • Technical skills relevant to your field
  • Soft skills (Communication, Leadership)
  • Tools and software proficiency
  • Languages spoken
  • Order by relevance to the position
