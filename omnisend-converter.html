📋 How to Use — Monthly Converter
1.In Omnisend go to Reports → Overview, set the date range to a single month, and copy the stats block shown on screen.
2.Enter the month label (e.g. Mar '26) in the field below, paste the data, and click Parse.
3.Repeat for each month you want to add — rows accumulate in the table below. Reorder or remove rows before downloading.
4.Click Download CSV when ready — the file matches template-monthly.csv exactly.
