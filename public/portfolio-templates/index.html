Frequently Asked Questions

Everything you need to know about our portfolio templates.

Do I need coding skills to use these templates? +

No coding skills required! Our templates come with simple editable files where you just replace the placeholder content with your information. The templates handle all the design and layout automatically.

How do I add my projects to the portfolio? +

Each template includes detailed instructions for adding projects. Typically you'll:

  1. Add your project images to the assets folder
  2. Edit the projects.json file (or similar) with your project details
  3. Optionally write case studies in the provided markdown files

No technical knowledge is needed - just follow the step-by-step guide included with your template.

Can I customize the colors and fonts? +

Absolutely! You can personalize your portfolio by editing the style.css file included with your template. Simply:

  1. Open style.css in any text editor
  2. Modify the color variables (like --primary-color) and font properties
  3. Save your changes and refresh your portfolio page

For advanced customization, you can edit any CSS rules in the file. No coding experience is needed for basic changes - just follow the clear comments we've included.

Example customization:

/* ======== Customize These Values ======== */
:root {
  --primary-color: #3a86ff;  /* Change main accent color */
  --text-color: #2b2d42;     /* Change text color */
  --font-main: 'Segoe UI', sans-serif; /* Change font */
}

All templates include detailed instructions in the Template_Guide.markdown file for common customizations.

What's included in the download? +

You'll receive a ZIP file containing:

  • All HTML, CSS, and JavaScript files
  • Project template files
  • Image placeholder assets
  • Detailed setup instructions
  • Customization guide
  • Tips for optimizing your portfolio
  • Free updates for one year
How do I host my portfolio? +

We provide detailed instructions for free hosting options like:

  • GitHub Pages (simple and free)
  • Netlify (with continuous deployment)
  • Vercel (great performance)

You can also host on any standard web hosting service. No special server requirements needed - these are static websites that work anywhere.
